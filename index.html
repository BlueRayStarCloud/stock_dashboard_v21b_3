☁️ Cloudflare Workers Proxy

透過 Cloudflare Workers 作為 CORS proxy，桌機與手機瀏覽器皆可正常使用，免費方案每日 10 萬次請求。

  1. dash.cloudflare.com → 登入（免費帳號即可）
  2. 左側選單 → Workers & Pages → 建立應用程式
  3. 選「建立 Worker」→ 點擊「部署」
  4. 部署後點「編輯程式碼」，貼入下方程式碼
  5. 點右上角「部署」→ 複製 Worker 網址貼到下方
Worker 程式碼（點擊複製）：
export default {
  async fetch(request) {
    const url = new URL(request.url);
    const action = url.searchParams.get('action') || '';
    const corsHeaders = {
      'Content-Type': 'application/json;charset=UTF-8',
      'Access-Control-Allow-Origin': '*'
    };
    // 股票搜尋自動完成
    if (action === 'search') {
      const q = url.searchParams.get('q') || '';
      const target = 'https://query1.finance.yahoo.com/v1/finance/search?q=' + encodeURIComponent(q) + '&lang=zh-TW®ion=TW"esCount=10&newsCount=0&enableFuzzyQuery=false"esQueryId=tss_match_phrase_query';
      const res = await fetch(target, { headers: { 'User-Agent': 'Mozilla/5.0', 'Accept': 'application/json' } });
      return new Response(await res.text(), { headers: corsHeaders });
    }
    // 歷史資料
    const stockNo = url.searchParams.get('stockNo') || '';
    const from = url.searchParams.get('from') || '';
    const to = url.searchParams.get('to') || '';
    const market = url.searchParams.get('market') || 'TW';
    const symbol = market === 'US' ? stockNo : stockNo + '.' + market;
    const target = 'https://query1.finance.yahoo.com/v8/finance/chart/' + symbol
      + '?period1=' + from + '&period2=' + to + '&interval=1d&includeAdjustedClose=true';
    const res = await fetch(target, { headers: { 'User-Agent': 'Mozilla/5.0', 'Accept': 'application/json' } });
    return new Response(await res.text(), { headers: corsHeaders });
  },
};
⚠️ 已更新上櫃支援，請重新複製程式碼並部署新版本 Worker。
